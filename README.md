# Task Management Dashboard

A simple task management dashboard built with **Angular** and **Angular Material**. Users can:

- View a list of tasks
- Add, edit, delete tasks
- Filter by status
- Sort by due date
- View a summary of tasks by status

---

## 📦 Tech Stack

- Angular 16+
- Angular Material
- RxJS for state management
- SCSS for styling
